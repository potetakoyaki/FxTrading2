# FX Strategy Doctor

MT4・MT5のトレード履歴CSVをアップロードするだけで、FX戦略を自動診断するWebアプリです。

## 機能

- **戦略スコア（100点満点）** - PF・RR・期待値・ドローダウンを総合評価
- **8つのパフォーマンス指標** - 勝率・PF・RR・期待値・純損益・最大ドローダウン・最大連敗数
- **弱点ランキング** - 通貨ペア別・時間帯別の弱点を自動検出
- **リスク診断（5項目）** - 3段階判定 + 診断解説 + アクション提案
- **モンテカルロシミュレーション** - 1,000回の試行による将来資金予測
- **レポート出力** - HTML・CSV形式でダウンロード
- **日本語・英語対応**

## 技術スタック

- React 19 + TypeScript
- Tailwind CSS 4
- shadcn/ui
- Recharts（チャート）
- PapaParse（CSV解析）
- Vite（ビルドツール）

## セットアップ

### 必要な環境

- Node.js 18以上
- pnpm（推奨）または npm

### インストール

```bash
# リポジトリをクローン（またはZIPを解凍）
cd fx-strategy-doctor

# 依存関係のインストール
pnpm install

# 開発サーバーの起動
pnpm dev
```

ブラウザで `http://localhost:3000` を開きます。

### ビルド（本番環境用）

```bash
pnpm build
```

`dist/` ディレクトリに静的ファイルが生成されます。

## デプロイ

生成された `dist/` ディレクトリの内容を任意の静的ホスティングサービスにデプロイできます。

- Vercel
- Netlify
- GitHub Pages
- AWS S3 + CloudFront
- その他の静的ホスティング

## 対応CSVフォーマット

- MT4 トレード履歴CSV（標準エクスポート形式）
- MT5 トレード履歴CSV（標準エクスポート形式）
