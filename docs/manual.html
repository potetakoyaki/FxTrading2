1. FX Strategy Doctor とは

FX Strategy Doctorは、MT4/MT5のトレード履歴を読み込み、35種類の分析パターンで戦略を多角的に評価するWebアプリケーションです。

主な機能:

  • 100点満点の戦略スコア — PF・RR・期待値・DDの4指標を重み付けして総合評価
  • 5項目のリスク診断 — DD・PF・連敗・期待値・勝率を各3段階（安全/注意/危険）で評価
  • 通貨ペア別・時間帯別・曜日別分析 — 弱い通貨ペアや時間帯を特定
  • モンテカルロシミュレーション — 1,000回の統計シミュレーションで破綻確率を算出
  • 改善提案 — 具体的なフィルター条件付きの改善アドバイス
  • レポート出力 — HTML/CSVでのレポートダウンロード
プライバシーについて アップロードされたファイルはサーバーに送信されません。すべての分析処理はお使いのブラウザ内で完結します。

2. 対応ファイル形式

形式拡張子説明
CSV.csvMT4/MT5の口座履歴をCSVエクスポートしたもの
Excel.xlsx / .xls / .xmlMT5のExcelレポート、またはCSVをExcelで保存したもの
HTML.htm / .htmlMT4/MT5の「詳細レポート」HTMLファイル
ファイルサイズ上限 最大50MBまでのファイルに対応しています。通常のトレード履歴であれば十分なサイズです。
注意: 文字エンコーディング MT5のHTMLレポートはUTF-16 LEエンコーディング（BOM付き）で出力されます。本ツールは自動的にエンコーディングを検出して処理するため、変換は不要です。

3. 基本操作 — ファイルのアップロードと分析

  1. ファイルを選択
    画面中央のアップロードエリアをクリックするか、ファイルをドラッグ＆ドロップします。対応形式（CSV / Excel / HTML）のファイルを選択してください。
  2. ファイルを確認
    選択したファイル名とサイズが表示されます。正しいファイルであることを確認してください。
  3. 「分析する」ボタンをクリック
    分析処理が開始されます。通常数秒で完了します。
  4. 分析結果を確認
    分析完了後、自動的に結果画面に遷移します。各セクションを確認してください。
サンプルデータで試す アップロードエリア下の「サンプルデータで試す」ボタンをクリックすると、デモ用のトレード履歴で分析を体験できます。

4. 分析結果の見方

4-1. 戦略スコア（Strategy Score）

100点満点で戦略の総合力を評価します。以下の4項目を重み付けして算出します。

項目配点評価基準
Profit Factor（PF）25点総利益 ÷ 総損失。1.5以上で高評価
Risk Reward（RR）25点平均利益 ÷ 平均損失。1.0以上で高評価
期待値25点1トレードあたりの期待利益。正の値で高評価
最大DD25点最大ドローダウン率。低いほど高評価
グレードスコア範囲評価
S85〜100非常に優秀な戦略
A70〜84優秀な戦略
B55〜69改善の余地あり
C40〜54要改善
D25〜39戦略の見直しが必要
F0〜24根本的な戦略変更が必要

4-2. パフォーマンス指標

  • トレード回数 — 分析対象の総トレード数
  • 勝率 — 勝ちトレードの割合（%）
  • Profit Factor — 総利益 ÷ 総損失。1.0以上で利益が出ている状態
  • Risk Reward — 平均利益額 ÷ 平均損失額
  • 期待値 — 1トレードあたりの期待利益（円またはドル）
  • 純損益 — 手数料・スワップ込みの合計損益
  • 最大DD — 資金ピークからの最大下落額と下落率
  • 最大勝ち / 最大負け — 1トレードでの最大利益と最大損失

4-3. リスク診断

5つのリスク項目それぞれを3段階（安全 / 注意 / 危険）で評価します。

診断項目安全注意危険
最大ドローダウン15%以下15〜30%30%超
Profit Factor1.2以上1.0〜1.21.0未満
最大連敗数5回未満5〜7回8回以上
期待値十分にプラス平均損失の10%未満マイナス
勝率45%以上30〜45%30%未満

4-4. 弱点ランキング

通貨ペア別・時間帯別の分析結果から、パフォーマンスが低い項目を自動検出し、重要度順にランキング表示します。「HIGH」は即座の対応が必要、「MEDIUM」は改善推奨を意味します。

4-5. 改善提案

35種類の分析パターンに基づき、具体的な改善アドバイスを提供します。各提案には優先度（高/中/低）とカテゴリ（リスク管理/エントリー/メンタル等）が表示されます。提案内容にはフィルター条件（例: 「金曜16時以降のトレードを避ける」）など、実行可能な具体策が含まれます。

5. 詳細分析パネル

各パネルはデフォルトで閉じた状態で表示されます。パネルヘッダーをクリックすると展開されます。

5-1. 曜日別分析

月曜〜金曜の各曜日ごとの勝率・PF・損益を集計します。特定の曜日にパフォーマンスが悪い場合、その曜日のトレードを避けることで全体成績が改善する可能性があります。

5-2. ロットサイズ相関

ロットサイズ別にトレード成績を分析します。ロットが大きいときに損失が増える傾向がないかを確認できます。

5-3. トレード品質分析

  • 利益集中率 — 上位20%のトレードが総利益の何%を占めるか。高いほど少数のトレードに依存
  • 損失集中率 — 上位20%の損失トレードが総損失の何%を占めるか。高いほど大きな損失が集中
  • 一貫性スコア — 集中率が低いほどトレード結果にバラつきが少なく、スコアが高くなります

5-4. 損益分布

全トレードの損益額をヒストグラムで表示します。利益と損失の分布形状から、戦略の特性（コツコツドカン型、スイング型など）を視覚的に把握できます。

5-5. 低品質トレード影響度

最も損失の大きいトレード（最悪トレード）を除外した場合の成績改善シミュレーションです。除外対象トレードの共通パターン（通貨ペア・曜日・時間帯）も分析され、具体的なフィルター提案に繋がります。

6. モンテカルロシミュレーション

過去のトレード結果をランダムに並べ替えて1,000回シミュレーションすることで、将来の資金推移の統計的な分布を予測します。

決定論的シミュレーション 同じトレードデータに対しては常に同じ結果が得られます（シード付き疑似乱数を使用）。読み込みのたびに結果が変わることはありません。
指標説明
利益確率1,000回のシミュレーション中、最終的にプラスになった割合
破綻確率資金がゼロ以下に到達した割合。シミュレーション長はトレード数の2倍（最低200回）
中央値リターンシミュレーション結果の50パーセンタイル
最大ドローダウン全シミュレーションの95パーセンタイルDD

エクイティカーブの読み方

チャートには複数のシミュレーションパスが表示されます。緑色の線は利益で終わったパス、赤色は損失で終わったパスを示します。パスの広がりが大きいほど、結果のバラつき（リスク）が大きいことを意味します。

8. MT4からの履歴エクスポート方法

  1. MT4を起動し、「ターミナル」ウィンドウを開く
    メニューバーの「表示」→「ターミナル」、またはキーボードの Ctrl + T を押します。
  2. 「口座履歴」タブを選択
    ターミナルウィンドウ下部の「口座履歴」タブをクリックします。
  3. 期間を設定
    口座履歴エリア内で右クリック → 「全履歴」を選択すると、全期間のトレード履歴が表示されます。特定期間を指定する場合は「期間のカスタム設定」を選択してください。
  4. レポートを保存
    口座履歴エリア内で右クリック → 以下のいずれかを選択:
    • 「レポートの保存」 → HTMLファイル（.htm）で保存
    • 「詳細レポートの保存」 → より詳細なHTMLファイルで保存（推奨）
  5. 保存したファイルをアップロード
    保存した .htm ファイルをFX Strategy Doctorにアップロードしてください。
注意: 「全履歴」の選択を忘れずに デフォルトでは直近の履歴のみ表示される場合があります。分析の精度を高めるために、必ず「全履歴」を選択してからレポートを保存してください。

9. MT5からの履歴エクスポート方法

  1. MT5を起動し、「ツールボックス」ウィンドウを開く
    メニューバーの「表示」→「ツールボックス」、またはキーボードの Ctrl + T を押します。
  2. 「口座履歴」タブを選択
    ツールボックスの「口座履歴」タブをクリックします。
  3. 期間を設定
    口座履歴エリア内で右クリック → 「期間」→「全履歴」を選択します。
  4. レポートを保存
    口座履歴エリア内で右クリック → 以下のいずれかを選択:
    • 「レポート」→「HTML」 → HTMLファイルで保存
    • 「レポート」→「Excel (xlsx)」 → Excelファイルで保存
    いずれの形式でも本ツールで読み込み可能です。

10. よくある質問（FAQ）

Q: データは外部に送信されますか？

いいえ。すべての分析処理はブラウザ内で行われます。トレードデータがサーバーに送信されることはありません。

Q: 同じファイルなのに結果が変わることはありますか？

いいえ。モンテカルロシミュレーションを含むすべての計算は決定論的（シード付き疑似乱数使用）です。同じデータであれば常に同じ結果が出ます。

Q: スマートフォンで使えますか？

はい。レスポンシブデザインに対応しており、スマートフォンやタブレットのブラウザからもご利用いただけます。ただし、PC環境での利用を推奨します。

Q: 対応しているブローカーは？

MT4/MT5の標準フォーマットに対応しているため、MT4/MT5を採用しているすべてのブローカーに対応しています。ヘッダー名のマッピングにより、各ブローカー固有の列名にも柔軟に対応します。

Q: EA（自動売買）の履歴も分析できますか？

はい。裁量トレード、EA、いずれの履歴も同じ形式で出力されるため、問題なく分析できます。

Q: 日本語と英語を切り替えられますか？

はい。画面右上の言語切替ボタンで日本語と英語を切り替えることができます。分析結果やアドバイスもすべて切り替わります。

Q: 手数料やスワップは計算に含まれますか？

はい。トレード履歴にCommission（手数料）、Swap、Taxes（税金）、Fee（費用）の列がある場合、自動的に損益に加算して計算します。

11. トラブルシューティング

症状原因と対処法
「必須列が見つかりません」と表示される CSVのヘッダー行に Time、Symbol、Profit に相当する列名がありません。MT4/MT5の口座履歴からエクスポートしたファイルであることを確認してください。手動で作成したCSVの場合、ヘッダー名が正しいか確認してください。
「N行のデータをスキップしました」と表示される 一部の行が解析できませんでした。入出金履歴やバランス調整行は自動スキップされるため、通常は問題ありません。スキップ数がトレード数に比べて多い場合は、ファイル形式を確認してください。
ファイルを選択できない 対応形式（.csv / .xlsx / .xls / .xml / .htm / .html）であることを確認してください。ファイルサイズが50MBを超えている場合はエラーになります。
分析結果が実態と異なる MT4の場合「全履歴」を選択せずに一部の期間のみエクスポートした可能性があります。口座履歴で右クリック →「全履歴」を選択してから再度エクスポートしてください。
文字化けする 本ツールはUTF-8、UTF-16 LE/BEエンコーディングに自動対応しています。それでも文字化けする場合は、ファイルをメモ帳で開き「UTF-8（BOM付き）」で保存し直してからアップロードしてください。
HTMLレポートが読み込めない MT4/MT5以外のツールで生成されたHTMLの場合、テーブル構造が異なる可能性があります。MT4/MT5の標準エクスポート機能を使用してください。
重要: 未決済ポジションについて MT4のHTMLレポートに「Open Trades」（未決済ポジション）セクションが含まれている場合、本ツールは自動的にこれを除外し、決済済みトレードのみを分析対象とします。含み損益は分析に含まれません。
