# MT4/MT5 取引履歴エクスポートガイド

FX Strategy Doctor で分析を行うために、MT4/MT5から取引履歴ファイルをエクスポートする手順です。

> **重要**: 履歴エクスポートには**パソコン版**のMT4/MT5が必要です（スマホ版にはエクスポート機能がありません）。エクスポート後の分析結果はスマホからでも確認できます。

## 1. MT4からのエクスポート（HTML形式）★おすすめ

HTML形式が最も情報量が多く、FX Strategy Doctorとの互換性が高いためおすすめです。

### 手順

1. **MT4を起動**し、ログインします。

2. **「口座履歴」タブを開く**
   - 画面下部の「ターミナル」ウィンドウ内にある「口座履歴」タブをクリックします。
   - ターミナルが表示されていない場合は、メニューバーの `表示` → `ターミナル`（またはショートカット `Ctrl + T`）で表示できます。

3. **表示期間を「全履歴」に設定**
   - 「口座履歴」タブ内の任意の場所を**右クリック**します。
   - コンテキストメニューから「全履歴」を選択します。

   > **注意**: デフォルトでは直近の取引しか表示されません。**必ず「全履歴」を選択**してください。分析対象の期間が増えるほど、より正確な診断が得られます。

4. **レポートを保存**
   - 再び「口座履歴」タブ内を**右クリック**します。
   - 「レポートの保存」を選択します。
   - 保存先フォルダとファイル名を指定して「保存」をクリックします。
   - `.htm` または `.html` 形式で保存されます。

5. **保存したファイルをFX Strategy Doctorにアップロード**
   - ブラウザでFX Strategy Doctorにログインし、保存したHTMLファイルをアップロードします。

## 2. MT4からのエクスポート（CSV形式）

CSV形式でもエクスポートできますが、HTML形式より情報が少ない場合があります。

### 手順

1. MT4で「口座履歴」タブを開きます（上記 手順1〜3と同じ）。

2. **「全履歴」を選択**した状態で、「口座履歴」タブ内を**右クリック**します。

3. 「レポートの保存」ではなく、以下のいずれかの方法でCSVを取得します:

   **方法A: コピー＆ペースト**
   - 「口座履歴」内を右クリック → 「コピー」を選択
   - Excel、Googleスプレッドシート、またはテキストエディタに貼り付け
   - CSVファイルとして保存

   **方法B: 詳細レポートから保存**
   - 右クリック → 「詳細レポートの保存」を選択
   - HTMLとして保存されますが、FX Strategy DoctorはHTML形式にも対応しています

## 3. MT5からのエクスポート（HTML形式）★おすすめ

### 手順

1. **MT5を起動**し、ログインします。

2. **「口座履歴」タブを開く**
   - 画面下部の「ツールボックス」ウィンドウ内にある「口座履歴」タブをクリックします。
   - ツールボックスが表示されていない場合は、メニューバーの `表示` → `ツールボックス`（またはショートカット `Ctrl + T`）で表示できます。

3. **表示期間を「全期間」に設定**
   - 「口座履歴」タブ内の任意の場所を**右クリック**します。
   - 「期間の設定」を選択します。
   - 表示されたダイアログで「全期間」を選択し、「OK」をクリックします。

   > **注意**: MT5では「全期間」を選択しないと直近の取引しか表示されません。必ず設定してください。

4. **レポートを保存**
   - 「口座履歴」タブ内を**右クリック**します。
   - 「レポート」→「HTML（ファイル名.html）」を選択します。
   - 保存先フォルダとファイル名を指定して「保存」をクリックします。

5. **保存したファイルをFX Strategy Doctorにアップロード**

---

## 4. MT5からのエクスポート（Excel形式）

MT5ではExcel形式での保存も可能です。

### 手順

1. MT5で「口座履歴」タブを開き、「全期間」を設定します（上記 手順1〜3と同じ）。

2. **「口座履歴」タブ内を右クリック**します。

3. 「レポート」→「Open XML（Excel）（ファイル名.xlsx）」を選択します。

4. 保存先フォルダとファイル名を指定して「保存」をクリックします。

5. `.xlsx` 形式で保存されます。そのままFX Strategy Doctorにアップロードできます。

## 5. スマホへのファイル転送方法

パソコンでエクスポートしたファイルをスマホに転送すれば、スマホからFX Strategy Doctorにアップロード・分析結果を確認できます。

### 転送方法

| 方法 | 手順 |
|---|---|
| **メール添付** | エクスポートしたファイルを自分宛にメール添付で送信し、スマホで受信して保存 |
| **クラウドストレージ** | Google Drive / Dropbox / iCloud Drive にファイルをアップロードし、スマホアプリから開く |
| **LINE / メッセンジャー** | 自分だけのLINEグループ（メモ用）にファイルを送信し、スマホで受信 |
| **USB / AirDrop** | iPhoneならAirDrop、AndroidならUSBケーブルで直接転送 |

### スマホからのアップロード手順

1. スマホのブラウザ（Chrome / Safari）でFX Strategy Doctorにログイン
2. 「ファイルを選択」をタップ
3. 転送したファイルを選択してアップロード
4. 分析結果が表示されます

## 6. トラブルシューティング

### 「口座履歴」にデータが表示されない

- **原因**: 期間設定がデフォルト（直近のみ）のままになっています。
- **対処**: 右クリック → 「全履歴」（MT4）または「期間の設定」→「全期間」（MT5）を選択してください。

### エクスポートしたファイルが空 / データが少ない

- **原因**: 「口座履歴」ではなく「取引」タブや「エクスポージャー」タブからエクスポートした可能性があります。
- **対処**: 必ず「口座履歴」タブから操作してください。「取引」タブは現在のポジション、「口座履歴」タブは決済済みトレードです。

### 「ファイル形式がサポートされていません」と表示される

- **対処**: 以下を確認してください:
  - 拡張子が `.htm`, `.html`, `.csv`, `.xlsx`, `.xls` のいずれかであること
  - MT4/MT5以外のツールからエクスポートしたファイルではないこと
  - ファイルが壊れていないこと（テキストエディタで開いて中身があるか確認）

### 分析結果の取引件数が実際より少ない

- **原因1**: 期間設定が「全履歴/全期間」になっていない可能性があります。
- **原因2**: 残高操作（入出金）のみの行が含まれ、実際のトレードが少ない可能性があります。
- **対処**: MT4/MT5で「全履歴」を選択し、再度エクスポートしてください。

### MT4/MT5のスマホアプリからエクスポートしたい

- **残念ながら、スマホ版MT4/MT5にはエクスポート機能がありません。**
- パソコン版のMT4/MT5でエクスポートしてください。
- パソコンが手元にない場合は、VPS上のMT4/MT5からエクスポートする方法もあります（VPSをご利用の場合）。

| プラットフォーム | 形式 | 拡張子 | おすすめ度 |
|---|---|---|---|
