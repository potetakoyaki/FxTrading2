### 価格戦略のポイント

- **お試しアカウント**: 7日間無料体験を用意し、購入ハードルを下げる
- **段階値上げ**: Brain/noteで「先着○名は特別価格」を活用
- **買い切り vs サブスク**: 初期は買い切り（無期限）で参入障壁を下げる

## Phase 5: 購入者対応フローの構築（Phase 3と並行）

### 5-1. アカウント発行手順

```
1. 購入通知を受信
2. 管理者パネル（/admin）にログイン
3. 「新規追加」→ ユーザーID・パスワード・メモ（購入者名+プラットフォーム）を入力
4. 利用期間を選択（買い切り → 無期限 / お試し → 7日）
5. ID/パスワードをコピーして購入者にメッセージで送付
```

### 5-2. テンプレートメッセージ

```
ご購入ありがとうございます！
以下のログイン情報でツールをご利用いただけます。

■ ツールURL: https://[your-domain]
■ ユーザーID: [ID]
■ パスワード: [PW]

操作マニュアル（PDF）も添付いたしますのでご確認ください。
ご不明点がございましたらお気軽にメッセージください。
```

### 5-3. サポート体制

- [ ] よくある質問（FAQ）ドキュメントの作成
- [ ] 「CSVの形式が合わない」系の問い合わせ対応テンプレート
