# FX Strategy Doctor 出品ロードマップ（コピペ出品用）

このドキュメントは、`FX Strategy Doctor` を複数プラットフォームへ出品するための実務用テンプレートです。
そのまま貼り付けて使える文面を中心に作成しています。

---

## 0. 前提（販売物の定義）

- 販売物: FX Strategy Doctor の利用権（購入者ID/パスワード発行型）
- 提供形式: Webサービス利用（ソースコード配布なし）
- 対象: MT4/MT5ユーザー（裁量・EAトレーダー）
- 対応ファイル: CSV / Excel（.xlsx/.xls） / HTML（.htm/.html）
- 主な機能:
  - 戦略スコア（100点）
  - 8指標（勝率 / PF / RR / 期待値 / 純損益 / 最大DD / 最大連敗 / トレード数）
  - 弱点分析（通貨ペア別・時間帯別）
  - リスク診断（5項目）
  - 1,000回モンテカルロシミュレーション
  - HTML/CSVレポート出力
  - 日本語/英語対応
- 商品画像素材: `docs/images/` 配下のスクリーンショット6点を各プラットフォームの商品画像として使用

### 1-10. FAQ（そのまま掲載可）

**Q. どのファイル形式に対応していますか？**
A. `CSV / Excel（.xlsx/.xls） / HTML（.htm/.html）`に対応しています。MT4/MT5の「口座履歴」からエクスポートしたファイルをそのままアップロードできます。

**Q. MT4とMT5の両方で使えますか？**
A. はい、どちらにも対応しています。

**Q. データは外部送信されますか？**
A. 分析はブラウザ内で完結する設計です。アップロードしたファイルはサーバーに送信・保存されません。

**Q. 日本語以外は対応していますか？**
A. 日本語・英語の表示切替に対応しています。

**Q. サブスクですか？**
A. いいえ。買い切り型です。一度ご購入いただければ、ずっとご利用いただけます。

**Q. 8指標とは具体的に何ですか？**
A. 勝率 / プロフィットファクター(PF) / リスクリワード比(RR) / 期待値 / 純損益 / 最大ドローダウン(DD) / 最大連敗 / 総トレード数 の8項目です。
